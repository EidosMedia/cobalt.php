cobalt.php
==========

[PHP 7.3 recommended for better performance](https://www.phoronix.com/scan.php?page=news_item&px=PHP-7.3-Performance-Benchmarks)

After PHP 7.3 and [composer](https://getcomposer.org/) installation:
````
$ cd <Cobalt PHP SDK>
$ composer install
````

To run all tests:
````
$ cd <Cobalt PHP SDK>
$ phpunit
````

To run a single test:
````
$ cd <Cobalt PHP SDK>
$ phpunit --filter test tests/<file.php>
````
